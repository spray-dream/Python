1. 环境搭建

  1. java sdk
    1. 安装
    2. 配置 JAVA_HOME 和 PATH
  2. Android SDK
    1. 解压
    2. 配置 ANDROID_HOME 和 PATH
  3. 虚拟机安装
    - 使用夜神模拟器
    • 也可以使用真机
      1. 手机通过usb线 连接到电脑
      2. 在手机上打开文件传输(通知栏，手机正在充电)
      3. 在电脑上会发现多了一个CD驱动器，里面是手机的驱动
      4. 安装驱动，电脑的设备管理器中就可以看到型号了
      5. 开启开发者模式： 手机 -- 设置 -- 关于手机 -- 版本号 (多次点击)
      6. 打开USB调试： 手机 -- 设置 开发者选项 -- usb调试(打开)
